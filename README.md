# ExtendDDG
An extension of AutoDDG

## Development

### Setup
```bash
uv sync
```

### Run
Test using the example Python script.
```bash
cd examples
uv run python quick_start.py
```

### Code Quality
```bash
uv run ruff check
uv run ruff format
```

## Requirements
- Python >=3.10
- [AutoDDG](https://github.com/VIDA-NYU/AutoDDG)
